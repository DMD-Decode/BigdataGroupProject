# 🌏 관광-환율 연관 분석 대시보드

한국관광공사의 관광 통계와 수출입은행의 환율 API 데이터를 기반으로, 관광객 수와 환율 간의 연관성을 시각화하고 분석하는 Streamlit 대시보드입니다.

## 주요 기능

- **🏠 메인 대시보드**: 최신 출국/방한 관광객 수 및 주요 4개국(USD, JPY, EUR, CNY) 환율 정보를 KPI로 제공하며, 월별 총 출입국자 비교 추이를 시각화합니다.
- **📈 환율-관광객 연관 분석**:
  - 아웃바운드(해외 출국) 또는 인바운드(방한 관광) 데이터와 특정 국가의 환율을 선택합니다.
  - **피어슨 상관관계 계수(Pearson Correlation Coefficient)**를 계산하여 두 변수 간의 상관관계를 수치화합니다.
  - 관광객 수(막대)와 환율(선)을 **이중 축 차트**로 시각화하여 변동 추이를 직관적으로 비교합니다.
- **👨‍👩‍👧‍👦 성별·연령별 분석**: 국민 해외관광(아웃바운드) 데이터를 성별과 연령대별로 필터링하여 월별 추이 및 연도별 히트맵을 제공합니다.
- **🛫 출국지별 분석**: 인천공항, 김해공항 등 출국 공항/항만별 월별 출국자 추이를 분석합니다.
- **🌏 방한관광(인바운드) 분석**: 방한 외래관광객 데이터를 성별, 연령, 방문 목적, 교통수단 등 상세 조건으로 필터링하여 월별 추이를 분석합니다.

## 🛠️ 기술 스택 및 라이브러리

- **Core**: Streamlit
- **Data Handling**: Pandas, NumPy
- **Visualization**: Plotly, Matplotlib, Seaborn
- **API**: Requests

## 🚀 실행 방법

### 1. 사전 준비

**A. API 키 설정**

이 프로젝트는 한국수출입은행의 환율 API를 사용합니다. 프로젝트 루트 디렉터리에 `.streamlit` 폴더를 생성하고, 그 안에 `secrets.toml` 파일을 생성하여 API 키를 추가해야 합니다.

- 파일 경로: `.streamlit/secrets.toml`
- 파일 내용:
  ```toml
  [api_keys]
  exim_bank = "여기에_발급받은_API_키를_입력하세요"
  ```

**B. CSV 데이터 파일**

분석에 필요한 관광공사 CSV 파일 2개를 `app.py`와 동일한 디렉터리에 다운로드해야 합니다.

1.  `한국관광공사_국민 해외관광객 월별 상세 집계.csv`
2.  `한국관광공사_방한 외래관광객 상세 월별 집계.csv`

**C. `requirements.txt` 파일**

`app.py`와 동일한 디렉터리에 `requirements.txt` 파일을 생성하고 다음 내용을 추가합니다.

```txt
streamlit
pandas
plotly
seaborn
matplotlib
requests
numpy
```
